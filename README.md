Цель - разработка программного обеспечения, позволяющего за счет оптимизации полигональной структуры трехмерных моделей сохранять их выразительность. Для достижения цели были изучены базовые методы оптимизации, проведен анализ конкурирующих аналогов и разработан собственный программный модуль.  
Оригинальность подхода состоит в применении алгоритмов оптимизации с выносом вычислений на графический процессор, что позволило добиться высокой производительности алгоритма при интерактивном взаимодействии с моделями в реальном времени. Разработанный продукт может быть применен для решения задач интерактивной визуализации, разработки приложений в области виртуальной реальности или дополненной реальности, построения LOD-уровней, а также в любой ситуации, когда возможно частичное ухудшения прорисовки с сохранением ее вполне приемлемого качества.
